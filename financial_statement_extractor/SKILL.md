# 财务报表章节全量解析器（状态机分组版）

## 描述

本技能用于对“财务报告章节”进行全量扫描，
自动识别所有财务报表，
并按报表标题进行分组。

⚠️ 本技能不再依赖单个报表页码范围，
只需要财务报告章节的起止页。

功能包括：

✅ 自动识别报表标题  
✅ 自动分组所有报表  
✅ 自动处理跨页  
✅ 自动避免母公司表混入  
✅ 提取完整逐行数据  
✅ 提取本期与上期金额  
✅ 识别单位信息  

---

## 调用方式

脚本文件名：skill.py  
函数名：extract_all_financial_statements

---

## 参数说明

- pdf_path: string，PDF文件路径
- finance_start_page: int，财务报告起始页
- finance_end_page: int，财务报告结束页

通常来自技能1的输出：

{
  "财务报告": {
    "start_page": 74,
    "end_page": 195
  }
}

---

## 返回值

JSON字符串，包含所有识别到的财务报表数据。
