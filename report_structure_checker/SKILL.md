# Skill: 报告结构与披露完整性检查器 (check_report_structure)

## 描述 (Description)
此技能用于检查上市公司年报的报告结构和披露完整性。它接收年报PDF文件的路径，以及来自年报索引的初步定位信息（例如：审计报告、重大事项、公司治理等章节的核心页段），然后根据这些信息，对关键披露项目进行初步的完整性检查，并报告定位状态。

## 参数 (Parameters)
### `pdf_path` (字符串, 必需)
需要分析的上市公司年报PDF文件的完整路径。

### `module_info` (字符串, 必需)
关于“报告结构与披露完整性”模块的定位信息，是JSON格式字符串，通常包含以下字段：
```json
{
  "核心章节": "审计报告, 公司治理, 重大事项",
  "核心页段": "2, 4, 28-29, 38, 42, 57-58, 67, 74-78",
  "审核对象": "审计意见类型、关键审计事项、关联交易披露、公司治理结构、内部控制评价",
  "审核动作": "提取、核对、对照、一致性审查",
  "预期输出": "披露项目清单、披露完整性评估、治理结构合规性清单"
}
